# 中文寫作防 AI 味速查表

## 五條核心規則

1. **刪填充短語** — 去掉開場白（「在當今...」）和強調拐杖詞（「值得注意的是」）
2. **打破公式結構** — 避免二元對比、戲劇性分段、修辭性鋪陳
3. **變化節奏** — 混合句長，兩項優於三項，段尾不要都是感嘆或總結
4. **信任讀者** — 直接講事實，跳過軟化、辯解、手把手引導
5. **刪金句** — 聽起來像可以被引用的句子，重寫它

## 個性與靈魂

無菌寫作跟 AI 生成一樣明顯。要有觀點、有節奏變化、承認事情的複雜性。
適當用「我」，允許一些不整齊，對感受要具體（不是「很感動」，是什麼感動）。

**Before：** 這次旅程讓我深刻體會到，真正的成長往往來自於走出舒適圈。每一步都充滿挑戰，但也因此收穫滿滿。
**After：** 第三天雨沒停過，鞋子全濕，我差點放棄。後來想想都走到這了，就繼續走。回來之後腳痛了一個禮拜，但確實比出發前更知道自己的極限在哪。

## 10 個常見結構 Pattern

1. **過度升華** — 不要把具體經驗硬拗成人生哲理
2. **宣傳腔** — 不要寫得像觀光手冊或新聞稿
3. **模糊歸因** — 引數據要具體，不要「專家認為」「研究顯示」
4. **三段式法則** — 不要什麼都硬分成三組
5. **否定式排比** — 「這不是 X，是 Y」用一次就夠了
6. **同義詞循環** — 對的詞重複用就好，不要硬換
7. **強行總結** — 「這條路雖然慢，但很穩」→ 砍掉
8. **公式化挑戰** — 「儘管...仍然...」→ 砍掉
9. **協作痕跡** — 「希望這對您有幫助」→ 刪除
10. **精緻比喻** — 如果像散文不像留言，就太雕了

## 台灣社群禁用句型

**禁用標點：** 破折號（——/—）、分號（；）、冒號宣告（「重點是：」）

| 類型 | 禁用範例 |
|------|----------|
| 假裝驚訝 | 說出來你可能不信 |
| 過度強調 | 這件事非常非常重要 |
| 空洞鋪陳 | 我發現一件事 |
| 冗餘確認 | 是的你沒看錯 |
| 誇張情緒 | 太棒了！真的太厲害！ |
| 數據宣告 | 結果很驚人： |
| 回音開場 | 「X」這句話超有感 |
| 空洞肯定 | 這種進度很實在 |

**禁止先複述對方內容再表示同意。** 直接從自己的觀點切入。

## 節奏檢查

- 混合短句（3-8 字）和長句（20+ 字），可以斷句
- 段落長度要變化，有的一句話就是一段，有的長一些
- 三句連續差不多長 → 打斷一句
