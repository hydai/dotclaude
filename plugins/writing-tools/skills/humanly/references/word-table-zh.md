# 中文 AI 廢話替換表

三級分類，依 AI 生成文本中的出現頻率判斷。

- **Tier 1 — 必換。** AI 極度愛用，看到就換。
- **Tier 2 — 聚集時才換。** 單獨出現 OK，同段落出現 2 個以上就是 AI 味。
- **Tier 3 — 高密度才換。** 正常詞彙，但 AI 用太多。整篇充斥時才處理。

## Tier 1 — 必換

| 替換 | 用 |
|---|---|
| 賦能 | 幫助、協助 |
| 打造 | 做、建立、開發 |
| 全方位 | 完整、各方面 |
| 落地 | 實行、執行、做出來 |
| 底層邏輯 | 基本原理、根本原因 |
| 顆粒度 | 細節程度、精細度 |
| 抓手 | 方法、著力點、切入點 |
| 拉齊 | 統一、對齊、同步 |
| 鏈路 | 流程、路徑、環節 |
| 護城河 | 優勢、門檻、競爭力 |
| 賽道 | 領域、方向、市場 |
| 深度（空洞修飾）| （刪除，或具體描述深在哪）|
| 此外 / 另外值得一提的是 | （刪除，直接說下一件事）|
| 至關重要 / 極其重要 | 重要、關鍵 |
| 不可磨滅 | 深刻、持久 |
| 作為...的證明 / 體現 | 說明了、顯示 |
| 標誌著 | （直接描述發生了什麼）|
| 見證了 | 看到了、經歷了 |
| 凸顯 / 彰顯 | 顯示、反映 |
| 為...奠定基礎 | 幫助、促成 |
| 充滿活力 | （具體描述活力在哪，或刪除）|
| 不斷演變的格局 | 在變的環境 |
| 深深植根於 | 來自、源於 |

## Tier 2 — 同段落出現 2+ 才換

| 替換 | 用 |
|---|---|
| 沉澱 | 累積、整理 |
| 迭代 | 改進、調整、改版 |
| 閉環 | 完整流程、從頭做到尾 |
| 心智 | 認知、印象、想法 |
| 觸達 | 接觸到、送到 |
| 降本增效 | 省錢提效、降低成本 |
| 生態 / 生態系 | 系統、環境、圈子 |
| 矩陣 | 分類表、組合 |
| 抓手 | 方法、工具 |
| 賦能（重複）| 幫助 |
| 增強 | 加強、改善 |
| 培養 / 促進 | 幫助、推動 |
| 強調（動詞）| （刪除，直接說重點）|
| 寶貴的 | 有用的、有價值的 |
| 格局（抽象名詞）| 環境、情況、局面 |

## Tier 3 — 高密度才換

| 詞 | 怎麼處理 |
|---|---|
| 重要的 / 非常重要 | 換成具體說明為什麼重要 |
| 顯著的 / 顯著地 | 給數字或比較 |
| 創新 / 創新性 | 描述新在哪裡 |
| 有效的 / 有效地 | 說怎麼有效，或引用數據 |
| 關鍵 / 關鍵性的 | 說明為什麼是關鍵 |
| 獨特的 | 描述獨特在哪 |
| 令人印象深刻 | 說什麼讓人印象深刻 |
| 值得注意的是 | （刪除，直接說）|
| 突出的 / 突出地 | 給具體事例 |
| 豐富的（比喻）| （具體描述豐富在哪）|
